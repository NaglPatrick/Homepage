“A Framework to Model Real-Time Databases” — now available on arXiv and ResearchGate!

🚀 What it’s about:
Real-time databases are used in high-stakes environments like air traffic control, finance, and process automation — where timing is everything.
But traditional modeling tools like UML fall short when it comes to handling time-sensitive data and dynamic transaction behavior.

📚 In this paper, I introduce:

  • A real-time object-oriented data model
  • A custom UML profile (UML-RTDB)
  • A working implementation using the Fujaba CASE tool
  • Code and SQL generation from model diagrams
  • A real-world case study in air traffic control
