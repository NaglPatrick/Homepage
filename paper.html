Paper Presentation

Rewritten Abstract:

Real-time databases are essential for managing time-sensitive data and executing time-constrained transactions in domains such as air traffic control and financial systems. Designing these databases demands innovative approaches to handle both the dynamic behavior of data and complex structural requirements. This paper explores key aspects of real-time data management, outlining the fundamental challenges and modeling requirements involved. We propose a comprehensive framework tailored to the needs of real-time systems, enhancing traditional database design through specialized tools and methods. To validate the framework, we include a case study that illustrates its practical application, demonstrating how real-world data can be modeled, and how system components like SQL queries and Java code can be effectively generated from the structural model.

Keywords:

Real-time data management, Dynamic database behavior, Software modeling tools, Time-sensitive applications, Case study validation
